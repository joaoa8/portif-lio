Conheça meu projeto de HTML

Neste projeto, construí a estrutura fundamental para o "Global Adventures", um blog de viagens fictício, utilizando exclusivamente HTML. O objetivo foi criar uma base de código limpa, semântica e pronta para a futura estilização com CSS e interatividade com JavaScript. A arquitetura do site foi desenvolvida para ser completa, incluindo uma página inicial com cabeçalho e navegação, páginas de detalhes para cada cidade com seções para conteúdo e multimídia, uma galeria de fotos e formulários de contato e inscrição. Em todo o processo, foi dada ênfase ao uso de tags semânticas do HTML5, como < header>, < nav>, < section> e < figure>, para garantir uma estrutura lógica, acessível e otimizada. Este trabalho demonstra minhas habilidades em HTML semântico, arquitetura de sites multi-página, organização de conteúdo e aplicação de princípios de acessibilidade web.

Conheça meu projeto de CSS

Neste projeto abrangente, desenvolvi uma página web completa e funcional para o site "World Trivia", aplicando um conjunto integrado de habilidades em HTML e CSS. A página foi construída com uma estrutura HTML semântica, dividida em seções claras para Início, Fatos de Trivia, Categorias, uma tabela de pontuações e um formulário de contato. Utilizei CSS Flexbox de forma extensiva para criar um layout moderno e responsivo, especialmente no menu de navegação e na organização do conteúdo. O projeto inclui a estilização detalhada de todos os elementos, desde a tipografia e cores globais até componentes complexos como a tabela de dados e o formulário de contato com labels acessíveis. Um destaque especial é o botão de inscrição, que possui uma animação de pulsação criada com @keyframes em CSS para atrair a atenção do usuário. Este trabalho demonstra minha proficiência na criação de interfaces web responsivas e acessíveis, combinando HTML estruturado com técnicas avançadas de CSS, como Flexbox e animações.

Sobre este linktree

Desenvolvi este portfólio no formato "linktree" como um projeto prático para aprender e aplicar os conceitos do framework Bootstrap. Utilizei suas classes de utilitários para construir um layout responsivo com Flexbox e para a estilização rápida de componentes, como os botões. O foco principal do aprendizado foi a implementação de funcionalidades interativas, explorando os componentes JavaScript do framework, como as janelas 'modal', que foram ativadas apenas com atributos data, sem a necessidade de código JS customizado. Este projeto serviu como uma excelente introdução ao desenvolvimento ágil com frameworks, resultando em uma página funcional que demonstra minha capacidade de aprender e aplicar novas tecnologias.
