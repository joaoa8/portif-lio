Conheça meu projeto de HTML e CSS

Como projeto final para concluir o curso "CSS Essentials" da plataforma Cisco NetAcad, desenvolvi a página "World Trivia" do zero, aplicando de forma equilibrada as competências de HTML e CSS. Fui responsável por toda a arquitetura de conteúdo, criando uma estrutura HTML semântica e acessível, que incluiu componentes como uma tabela de dados e um formulário de contato. Na parte de estilização, utilizei CSS para construir um design completamente responsivo com Flexbox, adicionei uma animação de pulsação com @keyframes para o botão de inscrição e cuidei de toda a identidade visual da página. Este projeto demonstra minha capacidade de realizar um trabalho completo de front-end, desde a fundação em HTML até a criação de uma experiência de usuário rica, interativa e visualmente agradável com CSS.

Sobre este linktree

Desenvolvi este portfólio no formato "linktree" como um projeto prático para aprender e aplicar os conceitos do framework Bootstrap. Utilizei suas classes de utilitários para construir um layout responsivo com Flexbox e para a estilização rápida de componentes, como os botões. O foco principal do aprendizado foi a implementação de funcionalidades interativas, explorando os componentes JavaScript do framework, como as janelas 'modal', que foram ativadas apenas com atributos data, sem a necessidade de código JS customizado. Este projeto serviu como uma excelente introdução ao desenvolvimento ágil com frameworks, resultando em uma página funcional que demonstra minha capacidade de aprender e aplicar novas tecnologias.
